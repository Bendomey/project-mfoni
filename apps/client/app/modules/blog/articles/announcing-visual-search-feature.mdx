Visual search is a powerful tool that allows you to find similar images based on a reference image. This feature is now available on mfoni, making it easier than ever to discover new content.

![Visual Search Feature](https://staging-images.mfoni.app/visual-search-feature.webp 'Visual Search Feature')

## Introduction

In this blog post, we will discuss the new visual search feature that has been added to our application. This feature is designed to improve user engagement and provide a more personalized experience.

## Feature Overview

The new visual search feature includes:

- **Face Scanning**: Users can take a selfie of themselves, and the application will use this image to find similar contents.
- **User-Friendly Interface**: The interface is designed to be intuitive and easy to navigate, making it accessible for all users.

## Benefits

The new visual search feature offers several benefits:

- **Enhanced User Experience**: Users can easily find their contents by taking a selfie of themselves, making it more relevant and engaging.
- **Increased Productivity**: By having quick access to the contents they need, users can work more efficiently.
- **Improved User Engagement**: This encourages users to interact with the application more frequently.

## How to Use the Feature

To use the new visual search feature, follow these simple steps:

1. **Using a desktop or laptop, open the application:**

   - Click on the "Visual Search" icon button located on the far end on the search bar.
   - Take a selfie using your device's camera.
   - The application will process the image and display similar contents based on your face.
   - Browse through the results and click on any content to view it in detail.

2. **Using a mobile device:**
   - Tap on the Search input bar.
   - Click on the "Visual Search" button.
   - Take a selfie using your device's camera.
   - The application will process the image and display similar contents based on your face.
   - Browse through the results and click on any content to view it in detail.
